## 已实现未上线（v0.2.4）

以下功能代码已完成，等待部署：

- **基金池重构**：基础倍率 1.0 + 常设奖励 0.5，加减关系；动态锁定(每日一次，24时重置)
- **标签提取**：正文 `#标签` 自动识别提取
- **获取记录**：star_income_history，仪表盘按钮查看所有星点来源
- **赛季系统**：15天赛季，自动开启/结算，软重置保留50%，赛季对比，"致未来自己"
- **连接力**：后台引擎，图密度算法，达到阈值(5/20/50)触发一次性星点奖励
- **跨界采集**：单篇≥2个不同领域标签，额外+5星点
- **连续加成**：7天+10% / 30天+30% / 90天+50%，取代旧惩罚机制
- **星点里程碑**：累计500/1000/2000/3000四档，解锁称号/倍率/输出下限/基金加成
- **兑换系统**：连续选择奖惩(fund+/coupon-)、镜像对比、消费/基金水位进度条
- **标签宇宙**：标签云(频次字号+连接度颜色)、排行榜、详情弹窗+关联笔记
- **三层反馈**：采集瞬间(行为确认+连接增量+相似笔记)、中观(能力面板+周期回顾)、宏观(长期推演+赛季结算)
- **4勋章**：灵光乍现、周常猎人、连线大师、连线新手(config表驱动)
- **配置体系**：defaults.yaml(模板) + config.yaml(运行时) + state.json(累积数据)

## 配置参数速查

> 权威来源是 `data/inspire/_狩猎系统/defaults.yaml` 和 `config.yaml`，此表仅作人类可读摘要。AI 接手时据此了解游戏数值体系。

### 采集与奖励

| 参数 | 值 | 说明 |
|------|-----|------|
| 基础星点 | 10 | 每次采集基础值 |
| 每日倍率 | 1.0 / 1.5 / 2.0 / 0.2 | 第1/2/3/4+条依次递减 |
| 跨界采集加成 | +5 | ≥2个不同领域标签时额外奖励 |
| 周回顾 | 200 | 次周提交 |
| 月度战报 | 500 | 次月提交 |
| 内容输出 | 750-1000 | 每次发布，最多5次/轮 |

### 连续加成

| 连续天数 | 奖励倍率 |
|---------|:---:|
| 7天 | +10% |
| 30天 | +30% |
| 90天 | +50% |

### 兑换与基金

| 参数 | 值 | 说明 |
|------|-----|------|
| 消费券汇率 | 1.0 | 1星点=1消费券 |
| 基金基础倍率 | 1.0 | 1星点存入=1基金值 |
| 基金常设奖励 | +0.5 | 有效基金倍率 1.5 |
| 基金最低提取 | 500 | 未达不可提取 |
| 基金锁定 | 每日一次 | 24时重置，非30天固定 |

### 连续选择奖惩（加减关系）

| 连续周数 | 基金 | 消费券 |
|:---:|:---:|:---:|
| 2周 | +0.05 (1.55) | -0.05 (0.95) |
| 4周 | +0.10 (1.60) | -0.10 (0.90) |
| 8周 | +0.15 (1.65) | -0.15 (0.85) |

### 连接力（后台引擎，前端不展示数值）

| 阈值 | 星点奖励 | 触发规则 |
|:---:|------|------|
| ≥5 | +50 | 每个阈值每赛季只触发一次 |
| ≥20 | +100 | |
| ≥50 | +200 | |

公式：`图密度 × 100 × log₂(节点数+1) × (跨界占比+0.1)`，范围 0.1~50

### 星点里程碑

| 累计星点 | 称号 | 效果 |
|:---:|------|------|
| 500 | 初露锋芒 | 称号解锁 |
| 1000 | 采集效率提升 | 第4条倍率 0.2→升级 |
| 2000 | 输出保障 | 内容输出下限 750→850 |
| 3000 | 兑换权益提升 | 基金常设奖励 +0.05 |

### 赛季

| 参数 | 值 |
|------|-----|
| 赛季长度 | 15天 |
| 赛季星点上限 | 3000 |
| 软重置比例 | 保留 50% |
| 自动开启 | true |

### 勋章（config 表驱动）

| ID | 名称 | 触发 |
|------|------|------|
| first_triple | 灵光乍现💡 | 单日采集≥3条(一次性) |
| weekly_hunter | 周常猎人🏹 | 周回顾连续≥4次 |
| link_master | 连线大师🕸️ | 月度跨界≥3次(一次性) |
| link_novice | 连线新手🔗 | 连接力≥5 |

## 技术栈

- 前端：Alpine.js（单页长滚动）
- 后端：FastAPI + uvicorn
- 存储：Markdown(灵感笔记) + YAML(配置) + JSON(状态)
- 部署：launchd + localtunnel（公网）
- 仓库：github.com/phyziix/hunterhunter
