## 已实现未上线（v0.24）

以下功能代码已完成，等待部署：

- **基金池重构**：基础倍率 1.0 + 常设奖励 0.5，加减关系；动态锁定(每日一次，24时重置)
- **标签提取**：正文 `#标签` 自动识别提取
- **获取记录**：star_income_history，仪表盘按钮查看所有星点来源
- **赛季系统**：15天赛季，自动开启/结算，软重置保留50%，赛季对比，"致未来自己"
- **连接力**：后台引擎，图密度算法，达到阈值(5/20/50)触发一次性星点奖励
- **跨界采集**：单篇≥2个不同领域标签，额外+5星点
- **连续加成**：7天+10% / 30天+30% / 90天+50%，取代旧惩罚机制
- **星点里程碑**：累计500/1000/2000/3000四档，解锁称号/倍率/输出下限/基金加成
- **兑换系统**：连续选择奖惩(fund+/coupon-)、镜像对比、消费/基金水位进度条
- **标签宇宙**：标签云(频次字号+连接度颜色)、排行榜、详情弹窗+关联笔记
- **三层反馈**：采集瞬间(行为确认+连接增量+相似笔记)、中观(能力面板+周期回顾)、宏观(长期推演+赛季结算)
- **4勋章**：灵光乍现、周常猎人、连线大师、连线新手(config表驱动)
- **配置体系**：defaults.yaml(模板) + config.yaml(运行时) + state.json(累积数据)
