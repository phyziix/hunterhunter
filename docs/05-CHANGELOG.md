## v0.22（开发中）

### Bug修复（2026-05-22）

  - 时间维度：消费即时无增值 vs 基金30天锁定预计增值

- **连续选择奖惩实现**：完成 `_calculate_exchange_rate()` 函数

  - 消费券连续2/4/8周：0.95x → 0.9x → 0.85x
  - 切换路径时 streak_weeks 归零
- **基金最低额度校验**：添加 500 星点最低兑换额度限制
- **创建 defaults.yaml**：包含完整初始配置（基础配置、基金配置、连续加成、路径奖惩、能力阈值、勋章配置、赛季配置）
- **勋章墙 undefined 修复**：修复 `icon` vs `emoji` 字段不匹配问题
- **基金加成显示修复**：后端 API 添加 `fund_bonus` 字段返回
- **汇率计算修复**：修复 `fund_base_rate` 字段未定义导致的 NaN 问题
- **灵感捕获面板按钮样式修复**：参考 v0.1 样式恢复捕获按钮样式
- **文本框 Ctrl+A 全选修复**：添加 `@keydown.meta.a.prevent` 支持 Mac Cmd+A
- **标签文本框无反应修复**：恢复 `@keyup.enter.prevent="addTag"` 事件绑定
- **提交按钮无法点击修复**：修复 `:disabled` 属性条件判断逻辑
- **勋章名称精简**：将"开拓者勋章"改为"开拓者"
- **规则速查汇率 NaN 修复**：修复 `fund_base_rate` 字段兼容性处理
- **赛季结算弹窗说明**：对应 PRD 7.3 宏观推演，赛季结束时自动弹出结算报告与"致未来自己"

### 开发进度

- **Phase 1 完成**（2026-05-22）：数据结构迁移与引擎核心重构
  - 完成 state.json 从 v0.1 到 v0.22 的自动迁移
  - 实现基金池与可用星点分离存储
  - 废除惩罚机制，改为连续天数递增奖励
  - 重写 exchange() 函数，支持路径选择逻辑
  - 新增 tag_graph 标签共现图数据结构
  - 完成 API 端点扩展，支持标签管理、兑换路径等

### 新增功能

- **能力值系统**：新增采集力、连接力、输出力三个能力值维度
  - 采集力：基于总笔记数、连续天数、活跃天数计算
  - 连接力：基于标签共现图密度、节点数、跨界笔记占比计算
  - 输出力：基于媒体发布次数、月度战报完成次数、内容输出总星点计算
  - 能力值达到阈值可解锁称号和特殊效果

- **标签宇宙**：新增标签云、标签排行榜、标签详情弹窗
  - 标签云按使用频次和连接度展示
  - 支持按标签查询关联笔记
  - 点击标签可跳转形成探索流

- **赛季系统**：新增 90 天赛季周期
  - 赛季主题（开拓者、连线大师、深度矿工、分享者）
  - 赛季结算报告（含"致未来自己"段落）
  - 赛季对比（本赛季 vs 上赛季）
  - 赛季专属勋章

- **兑换系统重构**
  - 基金池与可用星点分离存储
  - 每周路径选择（基金/消费券），不可跨周反悔
  - 连续选择奖惩机制
  - 镜像对比（核心干预机制）

- **连续加成**：废除惩罚机制，改为连续天数递增奖励
  - 7天：每日第1条采集额外 +10%
  - 30天：每日第1条采集额外 +30%
  - 90天：每日第1条采集额外 +50%

- **三层反馈体系**
  - 微观反馈：采集成功后展示三要素 + 相似笔记
  - 中观反馈：能力值面板 + 变更日志 + 周/月回顾数据叙事
  - 宏观反馈：长期推演仪表 + 赛季结算预测对比

### 技术变更

- **数据结构升级**：state.json 新增 15+ 字段
- **配置扩展**：config.yaml 新增连续加成、路径奖惩、能力值阈值、赛季配置
- **API 扩展**：新增标签、赛季、兑换路径等端点
- **算法优化**：连接力计算改用图密度算法

## v0.1（2026-05-20）

## 格式说明

### 版本号规则

- `vX.Y`：主版本号.X，次版本号.Y
- `vX.YZ`：X.Y 基础上的小修订 Z

### 变更类型

- **新增功能**：新增的功能或特性
- **改进优化**：对现有功能的改进
- **Bug修复**：修复的问题
- **技术变更**：底层技术实现的变化
- **文档更新**：文档的更新

### 撰写规范

- 每条变更简明扼要，说明"做了什么"
- 涉及用户可见的变化放在前面
- 技术细节放在后面
- 按日期倒序排列
