# 认知狩猎 - 变更日志

> 记录项目的版本变更历史，按日期倒序排列。

## v0.22（开发中）

### 开发进度

- **Phase 1 完成**（2026-05-22）：数据结构迁移与引擎核心重构
  - 完成 state.json 从 v0.1 到 v0.22 的自动迁移
  - 实现基金池与可用星点分离存储
  - 废除惩罚机制，改为连续天数递增奖励
  - 重写 exchange() 函数，支持路径选择逻辑
  - 新增 tag_graph 标签共现图数据结构
  - 完成 API 端点扩展，支持标签管理、兑换路径等

### 新增功能

- **能力值系统**：新增采集力、连接力、输出力三个能力值维度
  - 采集力：基于总笔记数、连续天数、活跃天数计算
  - 连接力：基于标签共现图密度、节点数、跨界笔记占比计算
  - 输出力：基于媒体发布次数、月度战报完成次数、内容输出总星点计算
  - 能力值达到阈值可解锁称号和特殊效果

- **标签宇宙**：新增标签云、标签排行榜、标签详情弹窗
  - 标签云按使用频次和连接度展示
  - 支持按标签查询关联笔记
  - 点击标签可跳转形成探索流

- **赛季系统**：新增 90 天赛季周期
  - 赛季主题（开拓者、连线大师、深度矿工、分享者）
  - 赛季结算报告（含"致未来自己"段落）
  - 赛季对比（本赛季 vs 上赛季）
  - 赛季专属勋章

- **兑换系统重构**
  - 基金池与可用星点分离存储
  - 每周路径选择（基金/消费券），不可跨周反悔
  - 连续选择奖惩机制
  - 镜像对比（核心干预机制）

- **连续加成**：废除惩罚机制，改为连续天数递增奖励
  - 7天：每日第1条采集额外 +10%
  - 30天：每日第1条采集额外 +30%
  - 90天：每日第1条采集额外 +50%

- **三层反馈体系**
  - 微观反馈：采集成功后展示三要素 + 相似笔记
  - 中观反馈：能力值面板 + 变更日志 + 周/月回顾数据叙事
  - 宏观反馈：长期推演仪表 + 赛季结算预测对比

### 技术变更

- **数据结构升级**：state.json 新增 15+ 字段
- **配置扩展**：config.yaml 新增连续加成、路径奖惩、能力值阈值、赛季配置
- **API 扩展**：新增标签、赛季、兑换路径等端点
- **算法优化**：连接力计算改用图密度算法

## v0.1（2026-05-20）

### 初始版本

- **灵感采集**：每日采集倍数（1.0x/1.5x/2.0x/0.2x），标签系统
- **任务与奖励**：周回顾（200星点）、月度战报（500星点）、内容输出（750-1000星点）
- **回顾与战报**：两步流程（生成素材 → 提交总结），自动添加 frontmatter
- **兑换系统**：消费券（汇率1.0）、基金（汇率1.5）
- **勋章系统**：灵光乍现、周常猎人
- **惩罚机制**：连续中断3天触发收益减半
- **UI界面**：灵感捕获、仪表盘、勋章墙、兑换中心、规则速查

## 格式说明

### 版本号规则

- `vX.Y`：主版本号.X，次版本号.Y
- `vX.YZ`：X.Y 基础上的小修订 Z

### 变更类型

- **新增功能**：新增的功能或特性
- **改进优化**：对现有功能的改进
- **Bug修复**：修复的问题
- **技术变更**：底层技术实现的变化
- **文档更新**：文档的更新

### 撰写规范

- 每条变更简明扼要，说明"做了什么"
- 涉及用户可见的变化放在前面
- 技术细节放在后面
- 按日期倒序排列
